== Description ==
This small plugin will stop members from sending email address to each other by removing the email address automatically 
when the message is sent .

== Installation ==

1. Install the plugin through the 'Plugins >> Add New' menu in WordPress or do a manual upload.

2. Activate the plugin

3. Thats it, sit back and relax.......
